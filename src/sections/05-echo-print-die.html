echo, print & die

echo & print

<?php

// echo — can take multiple arguments, returns nothing
echo 'Hello', ' ', 'World!';  // Hello World!

// print — takes ONE argument, returns 1 (usable in expressions)
print 'Hello World!';

// Common usage
$name = 'Alice';
echo "Welcome, {$name}!\n";
echo and print are language constructs, not functions. You should call them without parentheses. In practice, echo is used almost everywhere.

die & exit

<?php

// die() and exit() are identical — they halt execution immediately
$config = parse_ini_file('config.ini');

if ($config === false) {
    die('Error: could not load config file.');
}

// With an integer: sets the exit status code (0 = success)
exit(1);  // non-zero = error

// Often used for quick debugging (but use proper logging instead!)
die('Reached this point');

die is simply an alias for exit. Both accept an optional string message or int status code.
