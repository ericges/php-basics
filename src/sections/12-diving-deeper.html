Heredoc & Nowdoc

Heredoc and Nowdoc are a convenient way to write multi-line strings that can be used for formatting HTML, SQL queries, or other text content without having to escape special characters.

Heredoc
interpolates (like "")

$name = 'Alice';
$html = <<<HTML
    <h1>Welcome</h1>
    <p>Hello, {$name}!</p>
    HTML;

echo $html;

Nowdoc
literal (like '')


$code = <<<'TEXT'
    Variables like {$name} are
    NOT interpolated here.
    TEXT;

echo $code;

Heredoc behaves like double quotes; Nowdoc (note the single quotes around the tag) behaves like single-quoted strings.

The closing marker's indentation sets the baseline — that many leading spaces are stripped from every line.
This lets you indent heredocs naturally in your code without extra whitespace leaking into the output.

Source (indented in code)

echo <<<HTML
····<div>
········<p>Hello!</p>
····</div>
····HTML; /* <-- closing marker
^^^^ indents stripped in output */

Output (whitespace stripped)


<div>
····<p>Hello!</p>
</div>

If any line is indented less than the closing marker, PHP throws a parse error:

echo <<<HTML
··<p>Oops!</p>   // ← only 2 spaces
····HTML;        // ← closing marker expects 4
// Parse error: Invalid body indentation level

Docblocks

Docblocks are multiline comments that start with /**.
They describe code with annotations like @param, @return or @throws.
IDEs, static analysers, and documentation generators rely on them, e.g., to assist with code understanding and documentation.

<?php

/**
 * Calculate the total price including tax.
 *
 * @param  float  $price    Base price before tax
 * @param  float  $taxRate  Tax rate as a decimal (e.g. 0.19)
 * @return float            Final price with tax applied
 */
function totalWithTax(float $price, float $taxRate): float
{
    return $price * (1 + $taxRate);
}

/**
 * Summarize a list of items.
 *
 * @param  string[]  $items          List of item names
 * @param  int       $limit          Max items to display
 * @return string                    Comma-separated summary
 * @throws InvalidArgumentException  If $items is empty
 */
function summarize(array $items, int $limit = 5): string
{
    if (count($items) < 1) {
        throw new InvalidArgumentException('Items array cannot be empty');
    }

    return implode(', ', array_slice($items, 0, $limit));
}
