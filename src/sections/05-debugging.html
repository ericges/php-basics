Debugging Variables

Debugging Variables at Runtime

var_dump() — detailed type + value

$score  = 42;
$colors = ['red', 'green'];

var_dump($score);
var_dump($colors);
int(42)

array(2) {
  [0]=> string(3) "red"
  [1]=> string(5) "green"
}

var_dump() is the most informative tool — it shows the type and value of any variable, including nested structures.

Debugging Variables at Runtime

print_r() — human-readable, especially for arrays

$colors = ['red', 'green'];

// Echo directly:
print_r($colors);

// Or capture as string:
$output = print_r($colors, true);
Array
(
    [0] => red
    [1] => green
)

Pass true as the second argument to return the output as a string instead of printing it.

Debugging Variables at Runtime

gettype() — returns the type as a string

$val = 3.14;

echo gettype($val);

// Or capture as variable:
$type = gettype($val);
double

Useful for quick type checks. Note that floats report as "double" — a quirk inherited from C.
