Your First PHP File

The PHP-Tag

PHP code lives between an opening <?php and an optional closing ?> tag.

Without closing tag

<?php

echo 'Hello, world!';

Omit ?> where possible.

With closing tag

<?php

echo 'Hello, world!';

?>

Use ?> only for templating.

In code-only files (no HTML), start with <?php but omit the closing ?>. A stray newline after ?> can cause "headers already sent" errors. You will see ?> later when mixing PHP with HTML.

Create Your First File

Create a new file called hello.php and add the following:

<?php

echo 'Hello, world!';
PHP files use the .php extension. Save the file in any folder you like — you'll run it from there in the next step.

Running PHP

On the command line

php hello.php

Built-in development server

php -S localhost:8000

Opens a local web server in the current directory — no Apache/Nginx needed for learning.

With the server running, visit http://localhost:8000/hello.php to see your script's output in the browser.
