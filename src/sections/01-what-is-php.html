What is PHP?

PHP at a Glance

  • PHP: Hypertext Preprocessor — a recursive acronym
  • A server-side scripting language — your code runs on the server; the browser only receives HTML
  • Created by Rasmus Lerdorf in 1994; maintained by the PHP Group
  • Powers roughly 75 % of websites with a known back-end
  • We will use PHP 8.2 throughout this course

How PHP Works

  Browser                        Server
  ────────                       ──────
  GET /index.php        ──►     PHP interpreter executes the file
                                 └─► produces HTML output
                        ◄──     Sends the HTML to the browser

PHP is interpreted — the engine reads your .php file on every request and outputs the result.
